# 🎉 Birthday Website

A simple and responsive birthday webpage created using HTML and CSS. This project displays a birthday greeting, profile photo, age, and birthday date with a clean and colorful design.

## 🚀 Features

- Birthday greeting page
- Circular profile image
- Custom Google Fonts
- Responsive and centered layout
- Simple and beginner-friendly code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts

## 📂 Project Structure

```
birthday-website/
│
├── birthday.html
├── styles.css
└── image.jpg
```

## 📸 Preview

The webpage includes:
- Birthday title 🎉
- Profile picture
- Age display
- Birthday date

You can deploy this project for free using:

This project helped practice:
